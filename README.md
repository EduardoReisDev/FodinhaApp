# Fodinha :black_joker:
Aplicativo desenvolvido em Xamarin.Forms para contabilizar os pontos e auxiliar os jogadores no jogo de cartas Fodinha. :spades: :diamonds: :clubs: :hearts:

## Plugins e Framework
#### Plugins

- none

#### Framework
- Xamarin.Forms
- Xamarin.Essentials
